This asset include a full game. Here you can find a tools for creating "Hit the Mole" style game, on the example of our game "Croots Taper". This game works both on mobile phones and on computers. To switch control, you need to tick the variable "Mobile controll" on the object "TouchManager" on the scene.You can set the speed of movement of enemies, the time of appearance of the next enemy, points per click on the enemy and so on. All scripts contain comments. 

Package contains:
- full game;
- all sprites from our game;
- all animated sprites from our game;
- all scripts from our game;
- music and sound effects;
- all scripts contain comments
- test scene.
